# Core Banking Business System Technology

A comprehensive educational resource for understanding Core Banking Business System Technology - from fundamental concepts to advanced implementation patterns.

## Overview

This curriculum provides a structured learning path for understanding how modern core banking systems work. Whether you're a developer building banking applications, a product owner defining requirements, or a business analyst bridging the gap, these lessons will give you the knowledge needed to work effectively in banking technology.

## Glossary

| Term | Definition |
|------|------------|
| CBS | Core Banking System - centralized system managing banking operations |
| CIF | Customer Information File - unique customer identifier |
| GL | General Ledger - central accounting record |
| KYC | Know Your Customer - customer identification process |
| AML | Anti-Money Laundering - financial crime prevention |
| RTGS | Real-Time Gross Settlement - payment system |
| ACH | Automated Clearing House - batch payment network |
| SWIFT | Society for Worldwide Interbank Financial Telecommunication |
| IBAN | International Bank Account Number |
| EOD | End of Day - daily batch processing cycle |
| EOM | End of Month - monthly batch processing cycle |
